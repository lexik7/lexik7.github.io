CSS Loading Animation Demo

Learn how to build a purely CSS animation to add to any website!

In this tutorial, we will make a moving banner, text, and other elements using just CSS. Click here to see the full final product! Below is a preview:


Step 1: Simple HTML page setup

We must first start by creating a new webpage file with classes to separate the different elements that we will animate with CSS later on. Add the code below into a new "index.html" file: 

                
                    <div class="container">
                    <div class="split-screen"></div>
                    <div class="background-image"></div>
                    <div class="content">
                    <p>Hello<span class="comma"></span></p>
                    <p>World</p>
                    </div>
                    </div>

Step 2: CSS Styling

Next, we must style to webpage and its HTML that we just created. We will start with simple stylings and wait to style the animations later. Make a "style.css" page, and add the following code:

                
                    body {
                        margin: 0;
                        background-color: #b1bcbf;
                      }
                      .container {
                        width: 100%;
                        height: 100vh;
                        position: relative;
                      }
                      .background-image {
                        width: 50%;
                        position: absolute;
                        top: 0;
                        bottom: 0;
                        left: 0;
                        margin: auto;
                        background-image: url("https://images.pexels.com/photos/1261728/pexels-photo-1261728.jpeg");
                        background-size: cover;
                      }

Note that there is a background image that will serve as the animated image during the animation. This image can be customized and replaced by your own image. You can either "copy image address" of an image from the internet and replace the link above with your own, or tag your own image from your files.

Step 3: CSS Animations

To create the split-screen animation, we must recreate a certain animation timeline by adding timings for each animated element to create a smooth animation. Add the code below into your "style.css" page.

                
                    .split-screen {
                        width: 100%;
                        background-color: #2c2d2b;
                        position: absolute;
                        right: 0;
                        top: 0;
                        bottom: 0;
                        z-index: 2;
                      }

Next, we must define the animation using keyframes. Keyframes allow us to create specific timings throughout animations. For example, by setting certain properties for an element at one keyframe at the beginning of an animation, and then a second keyframe later throughout the animation with changed properties, the animation will shift between these properties for that element on a specific timeline.

First keyframe: setting the split screen from 100% to 50% of the page width:

            
                @keyframes reduceSize {
                    from {
                      width: 100%;
                    }
                    to {
                      width: 50%;
                    }
                  }

Assign Animation Keyframes to an Element

The animated keyframes we just coded above msut now be assigned to an HTML element. We can do this using the "animation-property" tag, and can control the animation timing using "animation-duration" and "animation-delay" properties. To make sure the animation remains in its final state without reverting back or restarting after its completion, we use the "animation-fill-mode" property:

                
                    animation-name: reduceSize;
                    animation-duration: 1.5s;
                    animation-delay: 0.5s;
                    animation-fill-mode: forwards;

You can add these properties under the ".split-screen" class in your css page. A shorthand notation for all four of these properties together could also be used instead:

                
                    animation: reduceSize 1.5s 0.5s forwards;

The Second Animation

The next animation will allow the split screen banenr to appear on move to the right of the screen. Add the following into your css page:

             
            @keyframes moveRight {
                from {
                  left: 0%;
                }
                to {
                  left: 50%;
                }
              }

Animating Page Content

Next, we must animate the content on the actual page. The three animations we will add are:

  • 1. Content will fade in while sliding up from the bottom of the page into the center
  • 2. Content will slide to the right of the page
  • 3. Content will change its text color from the original white to a background image

We must define all the animations and add the code into our css page:

            
                @keyframes fadeInUp {
                    from {
                      transform: translateY(100vh);
                      opacity: 0;
                    }
                    to {
                      transform: translateY(0);
                      opacity: 1;
                    }
                  }
                  @keyframes changeBackground {
                    to {
                      background-image: url("https://images.pexels.com/photos/1261728/pexels-photo-1261728.jpeg");
                      background-size: 200%;
                      background-position: center;
                      background-clip: text;
                      -webkit-background-clip: text;
                      color: transparent;
                    }
                  }
